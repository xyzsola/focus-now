chrome extension that blocks websites to increase focus mode productivity.

features:
- block selected websites by url (use regex)
- block all website except selected websites

ui:
- simple ui
- dark/light mode automatically based on system
- custom message when website is blocked
- custom image/gif/video when website is blocked
- timer to show how much time left to automatically unblock
- there is no option to unblock website manually
- block entire website (full screen)

extension settings:
- block selected websites by url (use regex)
- block all website except selected websites
- custom message when website is blocked
- custom image/gif/video when website is blocked
- block time (start and end time)
- block time by hours (how many hours to block)
